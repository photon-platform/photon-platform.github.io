The PHOTON platform is organized around numerous repositories that contain the code, configurations and documentation for the system.

  • .photon

  • .vim

  • logger

.photon

Tools, dot files, and configurations for the PHOTON platform.

  • docs: https://photon-platform.github.io/.photon

  • source: https://github.com/photon-platform/.photon

.vim

Configurations and plugins for the Vim text editor.

  • docs: https://photon-platform.github.io/vim

  • source: https://github.com/photon-platform/vim

logger

Tools to create and manage log entries in the reST format, specifically for the ABlog utility for Sphinx.

The app is built in Python using the new Textual UI library.

  • docs: https://photon-platform.github.io/photon-logger

  • source: https://github.com/photon-platform/photon-logger
